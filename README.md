Planning Poker® Desktop application built with Electron, React & Typescript

## Quick start

```bash
# clone repo
$ git clone https://github.com/lexmartinez/planning-poker

# open work folder
$ cd planning-poker

# install the dependencies
$ npm install

# run the app on develop
$ npm run dev
```
